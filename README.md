# Expense-Manager

## About / Synopsis

* Simple application that manages daily expenses of a person date wise for each month.
* Displays monthly and daily expenses summary
* Can add, replace and retrieve expenses
* Implemented using file Handling in C

## Table of contents

> * Each Month text file to store expenses and totals
> * C file containg code
